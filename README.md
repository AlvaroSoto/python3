# python3
Aprendizaje de Python desde cero.

## Índice de ejercicios

| # | Archivo | Tema | Descripción |
|---|---------|------|-------------|
| 01 | [01_print_input.py](01_print_input.py) | Print, variables e input | Uso de `print()`, declaración de variables, f-strings, captura de datos con `input()` y conversión de texto a número con `eval()` (enteros, decimales y expresiones). |
| 02 | [02_aritmetica.py](02_aritmetica.py) | Operadores aritméticos | Operaciones básicas entre dos números con `float()`: suma (`+`), resta (`-`), multiplicación (`*`), potenciación (`**`), división real (`/`), división entera (`//`) y módulo (`%`). El resultado muestra la operación completa (`a + b = resultado`). |
| 03 | [03_asignacion.py](03_asignacion.py) | Operadores de asignación compuesta | Uso de operadores aumentados (`+=`, `-=`, `*=`, `**=`, `/=`, `//=`, `%=`) con `float()`. Cada operación parte del mismo valor original para ver todos los resultados en una sola ejecución. |
| 04 | [04_funciones.py](04_funciones.py) | Funciones integradas de Python | Repaso de funciones built-in: `int()` (conversión y bases), `eval()` (evaluación de expresiones, advertencia de seguridad), `round()` (redondeo bancario), `abs()`, `max()`, `min()`, `pow()` (con módulo), `format()` (enteros, floats, alineación), `print()` (parámetros `sep` y `end`), `str()`, `chr()` / `ord()`, y operadores con cadenas (`+`, `*`, `len()`, `in`, indexación y slicing). |
